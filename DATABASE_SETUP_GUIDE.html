Complete guide to create and configure your database

Step 1: Create the Database

First, create a new database in phpMyAdmin called "tanglaw_lms"

  1. Open your browser and go to http://localhost/phpmyadmin
  2. Look at the left sidebar and click the New button (or icon to create a database)
  3. Enter database name: tanglaw_lms
  4. Choose collation: utf8mb4_unicode_ci (recommended)
  5. Click Create
‚úÖ Database "tanglaw_lms" created successfully

Step 2: Import the SQL File

Now import all tables, constraints, and sample data using the SQL file.

  1. After creating the database, you'll see it in the left sidebar
  2. Click on the database name: tanglaw_lms
  3. Click the Import tab at the top of the page
  4. Click Choose File and select: tanglaw_lms.sql
  5. Scroll to the bottom and click Import
‚úÖ All 11 tables created with sample data

Step 3: Verify Tables

Check that all tables were created successfully.

  1. In phpMyAdmin, click on your database: tanglaw_lms
  2. You should see 11 tables listed:
    • teachers
    • facilitators
    • detainees
    • subjects
    • grade_levels
    • providers
    • modules
    • activity_sheets
    • submissions
    • report_cards
    • distributions
  3. Click on each table to verify structure
‚úÖ All tables present and properly configured

Step 4: Run Database Setup Script (Optional)

Alternatively, you can use the PHP setup script if you prefer automatic creation:

  1. Open your browser to: http://localhost/xampp/htdocs/tanglaw/tanglawelearning/db_setup.php
  2. The script will automatically create all tables
  3. You'll see confirmation messages for each table
üí° Tip: Use the SQL import method (Step 2) if it's your first time. Use db_setup.php for quick refreshes.

Step 5: Start Using the System

Your database is ready! Now you can login to the LMS:

  1. Go to: http://localhost/xampp/htdocs/tanglaw/tanglawelearning/login.php
  2. Login with Admin credentials:
    • Username: admin
    • Password: admin123
    • Role: Administrator
  3. Create additional users (Teachers, Facilitators, Detainees)

‚ùå Troubleshooting

Problem: Import fails with "Unique constraint violation"

Solution: Drop all existing tables first, then re-import

Problem: Cannot find tanglaw_lms.sql file

Solution: The file should be in: c:\xampp\htdocs\tanglaw\tanglawelearning\tanglaw_lms.sql

Problem: phpMyAdmin not working

Solution: Start XAMPP Apache and MySQL services, then go to http://localhost/phpmyadmin
